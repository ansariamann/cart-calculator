# QuickMart - Smart Grocery List

QuickMart is a powerful and intuitive grocery list application that helps you create and manage your shopping lists with ease. It provides automatic calculations for totals, taxes, and discounts, making it an essential tool for budget-conscious shoppers.

## Features

- **Create and Manage Grocery Lists:** Easily add, edit, and remove items from your grocery list.
- **Automatic Calculations:** The app automatically calculates the total cost of your items, including taxes and discounts.
- **Budget Tracker:** Keep an eye on your spending with the built-in budget tracker.
- **Favorites:** Save your frequently purchased items for quick access.
- **Transaction History:** View your past shopping trips and expenses.
- **Camera Scanner:** Scan barcodes to quickly add items to your list.
- **PWA Support:** Install the app on your mobile device for a native-like experience.

## Technologies Used

This project is built with the following technologies:

- Next.js
- React
- TypeScript
- shadcn/ui
- Tailwind CSS

## Getting Started

To get started with the project, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/quickmart-grocery-app.git
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```

4.  **Build for production:**

    ```bash
    npm run build
    ```
